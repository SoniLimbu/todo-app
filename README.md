# To-Do App

A simple To-Do App built using HTML, CSS, and JavaScript.

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Save tasks using localStorage
- Responsive design

## Technologies Used
- HTML
- CSS
- JavaScript
